## Cheatsheet for Jupyter Notebook:

**Enter**                   : enter edit mode</br>
**Shift-Enter/Shift-Enter**		: run cell, select below</br>
**Ctrl-Enter/Ctrl-Enter**			: run cell</br>
**Alt-Enter/Option-Enter**		: run cell, insert below</br>
**A**					                : insert above</br>
**B**					                : insert below</br>
**D**					                : delete </br>
